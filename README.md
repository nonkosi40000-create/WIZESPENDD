WizeSpend: AI-Powered Financial Mentor
WizeSpend is a modern Android application built with Kotlin that transforms financial management into an engaging, habit-forming experience. Unlike traditional trackers, WizeSpend uses AI and gamification to move users from "mindless spending" to "wealth building."
🚀 Key Features
1. AI Coaching System
•
Actionable Insights: The AI Coach analyzes your spending data (Income vs. Expenses) and provides a step-by-step roadmap.
•
Natural Language Queries: Users can ask questions like "How can I save more?" and receive personalized advice based on their actual habits.
•
Risk Detection: The AI automatically identifies "At Risk" categories and suggests immediate corrective actions (e.g., "The 10% Rule").
2. Gamification & Streaks
•
Logging Streaks: Encourages daily consistency. A "First-Time" welcome message guides new users to start their first streak.
•
Dynamic Badges: Earn status badges like "Budget Pro" (Gold) or "New Starter" based on how well you stick to your goals.
•
Visual Reinforcement: High-energy emojis (🔥, ✨) celebrate progress and maintain user engagement.
3. Advanced Analytics & Data Capture
•
Multi-View Timeline: Toggle between Weekly, Monthly, and Yearly graphs.
◦
Weekly: Identifies weekend spending spikes.
◦
Monthly: Deep dive into category "leaks."
◦
Yearly: High-level risk assessment to prepare for expensive holiday seasons.
•
Visual Proof: Users can upload receipt photos, which are then displayed as thumbnails in category cards for easy verification.
🛠 UI/UX Design
•
Material Design 3: A professional, trust-evoking palette using deep blues and high-contrast status colors (Green for "Safe," Red for "At Risk").
•
Interactive Components: Floating cards, custom dropdowns, and a persistent bottom navigation bar for a seamless user journey.
📁 Technical Stack
•
Language: Kotlin
•
Persistence: SharedPreferences for fast, local data management.
•
Architecture: Activity-based navigation with dynamic UI state updates.
